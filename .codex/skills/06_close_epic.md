# Skill 06 — Close epic (status DONE + final doc sync)

## Developer prompt

Provide the EPIC-ID to close.

## Agent instructions

Read only:

- `docs/05-planning/epics.md`
- `docs/05-planning/epics/<EPIC-ID>/**`
- `checklists/pr.md`
- `checklists/doc-sync.md`

Do:

1) Verify all tasks and PR groups are DONE.
2) Update epic Status=DONE and add a short “delivered” note.
3) Ensure merge gates are satisfied (document intentional deviations).
