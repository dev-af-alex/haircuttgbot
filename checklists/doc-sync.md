- FR/scope → `docs/01-product/requirements.md`

- tech stack / toolchain → `docs/03-architecture/tech-stack.md`
- security behavior → `docs/02-nfr/security.md`
- performance assumptions → `docs/02-nfr/performance.md`
- reliability/ops → `docs/02-nfr/reliability.md`
- API contracts → `docs/03-architecture/apis.md`
- docker-compose run steps → `docs/04-delivery/local-dev.md`
- deploy steps → `docs/04-delivery/deploy-vm.md`
- epics/tasks status → `docs/05-planning/epics.md` and epic workspace
